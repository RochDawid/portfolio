About Me Dawid Roch

I define myself as a learning addict and I eager to contribute the world with quality software.

  • Back-End Development

Featured Project Verbenas App

This project was about creating an informative webpage for Majorcan people, mainly teenagers and youngsters, for knowing when would the local parties called 'verbenas' be celebrated and what would be done there. I designed the database and implemented it with MySQL, created a REST API for the data management with Node.js and began with the frontend web development with React implementing cards for each 'verbena' and a search bar with own logic management.

Messaging app Signal Clone

This project was about creating multiplatform clone of the well-known Signal messaging app, it was implemented with React Native and powered with Firebase for authentication and storage purposes.
