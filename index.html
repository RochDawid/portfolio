About Me Dawid Roch

Hi! I'm a 20 year-old Computer Science student in the University of the Balearic Islands. Currently I'm diving into web development exploring technologies such as HTML, CSS, JavaScript and it's frameworks both for Front-End and Back-End development (React, Angular, Node...). This environment gets me hyped up a lot and I would like to begin my professional career on this path although many other fields like Data Science are also of my interest. I define myself as a learning addict and I eager to contribute the world with quality software.

  • Front-End Development

    Currently I feel quite comfortable working around with HTML, CSS and JavaScript. I like to play around with some frameworks like React or Angular and I am used to manage forms, retrieving data from APIs and passing props between components.

  • Back-End Development

    With diverse experience on programming languages such as C, Java, JavaScript, AJAX and database design and administration knowledge (mainly relational databases with SQL) I am able to design and implement the software architecture desired for most web or mobile applications.

  • Data Science

    I have experience with R and Python and good math fundamentals (some math subjects passed with honors). In the university I worked together with Vodafone and Verne Technology Group where I managed and analyzed occupation data during the pandemic.

Featured Project Verbenas App

Informative webpage addressed to Majorcan teenagers and youngsters for knowing when would the local parties called 'verbenas' be celebrated and what would be done there. I designed the database and implemented it with MySQL, created a REST API for the data management with Node.js and began with the frontend web development with React implementing cards for each 'verbena' and a search bar with own logic management.

Messaging app Signal Clone

Multiplatform clone of the well-known Signal messaging app, it was implemented with React Native and powered with Firebase for authentication and storage purposes.

Videogame Database VGDB

Videogame database made with Angular 12 and using the free-to-use RAWG API for retrieving data. Implemented a dynamic search bar and a card based design. This project helped me understand and practice observables, angular's HttpClient module, using a loading service, working with material design dialogs and do a skeleton loading component which makes the user perceive a faster website.
